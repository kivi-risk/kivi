# 逐步回归法

> 逐步回归法是一种用于选择回归模型中变量的方法。它通过逐步添加或删除变量来找到最优的回归模型。

## 单变量回归

*批量进行单变量回归评估*

> `FeatureEvaluate`参数



```python
from kivi.feature import *
from kivi.datasets import *

df_bank = Dataset.bank_data()
df_bank = df_bank.select_dtypes(['int64', 'float64']).copy()

feature_eval = FeatureEvaluate(df=df_bank)
df_ans = feature_eval.stats()
print(df_ans.to_markdown())
```
